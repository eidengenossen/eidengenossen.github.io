About Warey

Warey is a Linux distribution that was made following certain objectives fixed for my convenience, which are:

    A unique 3 CPU architecture support for both 32-bits CPUs and 64-bits ones, which are x86, ARM (aarch) and RISC-V.

    Three models:

      A rolling release model, named mintwater;

      An experimental garage, named inksnow;

      A long-term support version, named goldfire.

    7 Desktop Environment built into the iso file:

      Spearmint: for casual users, contains the essential spear-utils;

      Rainwater: for developers and coders, contains the coding swiss army knife rain-utils;

      Snowfall: for enthusiasts that don't like to do everything themselves, but want to experiment with their DE, contains the intuitive fall-utils;

      Penink: for artists, contains the creative pen-utils;

      Goldheart: for user-friendliness, contains the highly well documented heart-utils;

      Rubyfire: for gamers, contains the optimized ruby-utils;

      Lite: for customizibility and minimal setup, the most linux-like experience that like DIY things, contains the minimal w-tools and w-utils.

    One and only one Package Manager: wimp (e.g: eidengenoosen@warey-btw:~$ sudo wimp -u install -y wareidgear)

    A Tiling Window Manager: wearim, which has two modes:

      Normal mode (like hyprland);

      Horizon mode (unlocks horizontal scroll, like the one in niri).

    Supports Wayland and Wareidgear support;
